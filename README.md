# Mi App de Expo

Esta es una aplicación de React Native creada con Expo SDK 54.

## Instalación

```bash
npm install
```

## Ejecutar la app

```bash
npm start
```

Luego escanea el código QR con tu app de Expo Go (SDK 54) en tu teléfono.

## Comandos disponibles

- `npm start` - Inicia el servidor de desarrollo
- `npm run android` - Abre en Android
- `npm run ios` - Abre en iOS
- `npm run web` - Abre en navegador web
